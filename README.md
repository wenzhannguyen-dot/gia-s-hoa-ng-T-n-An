# Multi-Timezone Digital Clock Project

## Overview
This project aims to create a multi-timezone digital clock that can display the current time in various timezones simultaneously. It's designed for use in applications where users are located in different parts of the world.

## Features
- Displays current time for multiple timezones.
- User-friendly interface.
- Responsive design suitable for various devices.
- Option to add or remove timezones dynamically.

## Installation
1. Clone the repository:
   ```
   git clone https://github.com/wenzhannguyen-dot/gia-s-hoa-ng-T-n-An.git
   ```
2. Navigate to the project directory:
   ```
   cd gia-s-hoa-ng-T-n-An
   ```
3. Install dependencies (if any):
   ```
   npm install
   ```

## Usage
- Open the `index.html` file in a browser to view the clock.
- Follow the instructions on the interface to add or configure timezones.
